# Zapret 2 (Запрет обход блокировки Дискорда и Ютуба)
<img width="980" height="519" alt="изображение" src="https://github.com/user-attachments/assets/8daab137-9c42-4171-854b-23eb3d8f5db4" />

Консольная CLI-версия Запрета 2 с возможностью добавление и создания своих собственных пресетов в один клик. Не содержит никаких лишних `exe` файлов, полностью открытый исходный код для параноиков и автономный режим без лишней сетевой активности.

> [!NOTE]
> Эта версия специально создавалась для поддержки Windows 7+ (_включая 32-битную_) и выше!
> Из-за 32 битной архитектуры производительность может быть слегка снижена.

## Скачивание

Перейдите в раздел [Releases](../../releases/latest) и скачайте один из файлов:

| Файл | Описание |
|------|----------|
| `zapret2-youtube-discord-vX.X.X.exe` | **Рекомендуется.** Самораспаковывающийся архив — просто запустите и выберите папку для распаковки. Работает без установленных архиваторов. |
| `zapret2-youtube-discord-vX.X.X.7z` | Архив 7z — нужен [7-Zip](https://7-zip.org/) для распаковки. |

> [!IMPORTANT]
> **Почему нет ZIP?** Встроенный распаковщик Windows ломает **кириллические имена файлов** в ZIP-архивах — пресеты вроде `Ростелеком.txt` превращаются в `���⥫����.txt` и не работают. Это известный баг Windows: ZIP хранит имена в UTF-8, а Explorer читает их через системную кодировку. Формат **7z** и **SFX EXE** (основан на 7z) корректно обрабатывают кириллицу на всех системах.

## Быстрый старт

### 1. Распакуйте архив
- **EXE**: запустите скачанный `.exe` файл, выберите папку (например, Рабочий стол) и нажмите Extract
- **7z**: распакуйте через 7-Zip в любую папку

### 2. Запустите
Откройте распакованную папку и запустите один из файлов **от имени администратора** (правой кнопкой → "Запуск от имени администратора"):

| Файл | Для кого |
|------|----------|
| `zapret-console.bat` | **Для быстрого старта.** Выберите номер пресета — и готово. |
| `service.bat` | **Полное управление.** Автозапуск, диагностика, тестирование пресетов, очистка кэша Discord. |

> [!WARNING]
> **Не запускайте `winws2.exe` напрямую!** Он ничего не сделает без пресета. Всегда используйте `zapret-console.bat` или `service.bat`.

### 3. Выберите пресет
При запуске вы увидите пронумерованный список пресетов. Введите номер и нажмите Enter. Если не знаете какой выбрать — начните с **Default v5**.

### 4. Проверьте работу
Откройте Discord или YouTube. Если не работает — попробуйте другой пресет. Можно запустить автоматическое тестирование всех пресетов через `service.bat` → пункт 9.

## Основные возможности
- Свыше 70 встроенных пресетов
- Автоматическое тестирование пресетов
- Автозапуск вместе с системой (через Планировщик задач)
- Скрытый режим работы (без окна в панели задач)
- Диагностика конфликтов (VPN, антивирусы, другие обходы)
- Очистка кэша Discord
- Поддержка Windows 7+ (32-bit и 64-bit)

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Ничего не происходит при запуске | Запустите **от имени администратора**. Не запускайте `winws2.exe` напрямую — используйте `zapret-console.bat` или `service.bat` |
| "The system cannot write to the specified device" | Обновите до последней версии — ошибка исправлена |
| Discord/YouTube не работает | Попробуйте другой пресет. Запустите тестирование в `service.bat` → пункт 9 |
| Ошибки сертификатов после перезагрузки | Обновите до последней версии — добавлена автоочистка WinDivert |
| winws2 висит в панели задач | Переустановите автозапуск в `service.bat` → пункт 2 (новая версия использует скрытый режим) |
| Не работает на Windows 7 | Убедитесь что используете последнюю версию. Если winws2 запускается и сразу пропадает — обратитесь в Telegram |

## Структура файлов

```
zapret2-youtube-discord/
  zapret-console.bat    ← Быстрый запуск (выбор пресета)
  service.bat           ← Полное управление
  exe/
    winws2.exe          ← Движок обхода (НЕ запускать напрямую!)
    WinDivert*.dll/sys  ← Драйвер фильтрации пакетов
  presets/              ← Пресеты (70+ конфигураций)
  lists/                ← Списки IP-адресов и доменов
  lua/                  ← Lua-скрипты конфигурации
  utils/                ← Утилиты (тесты, автозапуск)
```
